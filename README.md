# CFG-MP and CFG-MP+ for SD3.5
This project implements CFG-MP and CFG-MP+ based on Stable Diffusion 3.5.

## Key features
- Using a forward-backward scheme for the manifold projection (CFG-MP)
- Using Anderson Acceleration to further improve the performance of this forward-backward scheme (CFG-MP+)

## Usage
Run generation using a single GPU:

```bash
python demo_SD.py
```
